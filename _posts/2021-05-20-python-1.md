### 縮排與區塊結構

Python 的程式敘述以行為單位，用空格和縮排區分區塊結構（迴圈、條件句等）。
<!--8.5 節詳細說明-->

Python 允許使用任意數量的空格或 Tab 來縮排（但同一層的區塊需使用相同縮排），而官方建議使用 **4 個空格**來縮排。**注意 Tab 與空格屬於不同縮排，不可混用**。

### 註解

大多數 Python 中的 `#` 符號後面的內容都是註解，會被解譯器忽略，若在字串中，則只是一個字元。

**請養成添加註解的好習慣，註記程式的目的，以便未來自己或他人能看懂。**

### 變數及其指派（assignment）

Python 的變數無需事先宣告型別，首次指派值給變數時，該變數就會自動建立。變數建立後可使用 `del 變數` 刪除。

與大部分語言的變數只能儲存宣告時的型別不同，可以指派任何型別給 Python 的變數，如：

```py
>>> x = "Hello" # 建立字串物件 "Hello"，並讓 x 參照到該物件
>>> print(x)
Hello
>>> x = 5 # 建立整數物件 5，並讓 x 參照到該物件 
>>> print(x)
5
```

Python 的變數型別會依據參照（reference）到的物件而定，但若濫用此特性則可能造成混淆。

#### Python 變數的運作方式

許多程式語言的變數可以視為儲存一個值的容器，但 Python 的變數則是參照到某個物件。

任意數量的變數可以參照到同一物件，當此物件發生更改，這些變數所參照的值也會更改：

```py
>>> a = [1, 2, 3] # list 詳細使用方法請見第 2 篇
>>> b = a
>>> c = a
>>> b[1] = 5
>>> print(a, b, c)
[1, 5, 3] [1, 5, 3] [1, 5, 3]
```

若 Python 變數參照的是不可變更的物件，當更改變數的值時，Python 會依據新值建立新物件參照到該物件：

```py
>>> a = 1 # 建立整數物件 1 並讓 a 參照（整數不可變）
>>> b = a 
>>> c = b
>>> b = 5 # 建立新整數物件 5，並參照到此物件
>>> print(a, b, c)
1 5 1
```

**Python 變數的概念類似於標籤而非容器，`=` 並不會把值存入變數中，因此不能儲存數值或資料**，在 Python 中，所有事物都是物件。

#### 變數的名稱

Python 的變數區分大小寫可包含英文、數字及底線 `_`，但不可以數字開頭。

> `_` 開頭或結尾的變數通常有特殊用途，建議命名變數時不以底線開頭或結尾。

### 運算式

即使運算式所有數值都是整數，結果也不一定為整數，除法會回傳一個浮點數。若要用整數除法回傳小於商的最大整數，可使用 `//`。除了數值，運算式也可以處理如字串、布林值等其他型別。

### 字串

Python 與多數語言相同，用雙引號 `"` 表示字串，也可以用單引號 `'` 代替。

反斜線 `\` 可用於跳脫字元，`\n` 代表換行符號、`\t` 代表 Tab 字元、`\\` 代表普通的反斜線、`\"` 則是普通的雙引號。

在單引號的字串中雙引號不需使用反斜線，反之亦然。若需要將字串拆成多行，則必須使用三引號 `"""` 或 `'''`，且字串裡的單引號與雙引號不需使用反斜線：

```py
x = """在跨行字串中，
使用 " 與 '
不須加上反斜線
"""
```

### 數字

Python 提供四種數字類型：整數、浮點數、複數、布林值。

- 整數位數無上限，僅受機器資源限制
- 浮點數可用小數點或科學記號表示，如：3.14159、-2e-8。精度由底層機器控制，通常為雙精度浮點數（double）
- 布林值只有 True 與 False，分別等於 1 與 0

`int()` 會捨去小數點後的部分，`float()` 則是把整數轉為浮點數。

#### 複數

Python 用 `nj` 表示複數虛部，n 可以是整數或浮點數，j 等於 \\(\sqrt{-1}\\)。複數運算結果以括號顯示，表示單一物件。

1j * 1j 會得到 -1，但仍會被視為複數物件（-1 + 0j），複數永遠不會自動轉換為整數或浮點數。可以使用 `real` 或 `imag` 屬性取得實部與虛部，結果以浮點數呈現：

```py
>>> z = (3 + 5j)
>>> z.real
3.0
>>> z.imag
5.0
```

### None 值

None 用於表示空值，若 Python 函式無回傳值，則預設會回傳 None。None 也可做為佔位符號，先保留該位置，之後再填值。

整個 Python 系統中只有一個 None 物件，意味著所有對 None 的參照都指向同一物件。若使用 `==` 比較，None 也只等於自身。

### `input()`

`input()` 函式可以取得使用者的輸入，也能使用提示字串：

```py
>>> job = input("輸入職業 ")
輸入職業 學生
>>> print(job)
學生
```

使用者的輸入為字串型別，若要作為數字使用，必須使用 `int()` 或 `float()` 轉換：

```py
>>> age = int(input("輸入年齡 "))
輸入年齡 24
>>> print(age + 1)
25
```

### Python 風格與命名範例（可略過）

Python 有一些建議的程式撰寫風格範例，記載於 [PEP 8](https://legacy.python.org/dev/peps/pep-0008/)（Python Enhancement Proposal）中，遵循 PEP 8 慣例能保持一致性，讓他人更容易理解你的程式碼。

Python 命名慣例：

- 模組 / 套件名稱：短名詞，全小寫，必要才使用底線（`_`）
- 函式名稱：全小寫，為增加可讀性可使用底線。例：`my_func()`
- 變數名稱：全小寫，為增加可讀性可使用底線。例：`my_var`
- 類別名稱：每個英文單字低一個字母大寫。例：`MyClass`
- 常數名稱：全大寫，單字間加底線。例：`TAX_RATE`
- 縮排：每級 4 個空格，不使用 Tab
- 比較：無須用 `==`　比較 True 或 False。例：`if not my_var():`

#### 底線開頭或結尾的名稱

- 前單底線 `_var`
  - 模組的私有變數或函式，用 `*` 匯入模組時無法匯入 `_` 開頭的名稱 <!-- 見第 10 章 -->
  - 類別或物件的私有變數或 method，外部仍可存取<!-- 見第 15 章 -->
- 前雙底線 `__var`
  - 類別或物件的私有變數或 method，外部無法直接使用原本名稱存取 <!-- 見第 15 章 -->
- 前後雙底線 `__var__`
  - 保留給 Python 內部使用的名稱 <!-- 見第 17 章 -->
- 後單底線 `var_`
  - 避免與 Python 關鍵字衝突
- `_`
  - 暫時性或不重要的資料 <!-- 見第 5 章 -->
